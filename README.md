# 🏄‍♂️ Quick Start

> clone/fork:

```bash
git clone https://github.com/chrontast/MultichainDashboard.git
```


cd MultichainDashboard
yarn install
yarn start



📝 Edit your frontend `Home.jsx` in `packages/react-app/src/views`

📱 Open http://localhost:3000 to see the app
